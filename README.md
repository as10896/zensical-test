# Zensical Test

## Local Preview

The fastest way to preview this site locally is with Docker Compose. If you have Docker Compose installed, simply run:

```sh
docker compose up
```

This starts the Zensical preview server using the official Zensical Docker image, so you don't need to install anything else.

### Alternative: Install Zensical directly

If you prefer not to use Docker, you can install Zensical with a package manager such as `uv` or `pip`. See the official documentation for installation instructions: [https://zensical.org/docs/get-started/](https://zensical.org/docs/get-started/)

Once installed, run the preview server with:

```sh
zensical serve
```
